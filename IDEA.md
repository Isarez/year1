# ไอเดียเกมใหม่

ไฟล์นี้ใช้เก็บไอเดียเกม/หมวดใหม่ที่ยังไม่ได้ทำ

**กติกา** (ดู [CLAUDE.md](CLAUDE.md) ด้วย):
- อ่านไฟล์นี้เฉพาะตอนจะเริ่มทำเกมใหม่เท่านั้น
- เมื่อนำไอเดียไหนไปทำเป็นเกมจริงแล้ว ให้ลบไอเดียนั้นออกจากไฟล์นี้ทันที
- แต่ละไอเดียเก็บเป็น 3 ส่วนเสมอ: **หัวข้อเกม** (ชื่อสั้นๆ ใช้เป็น `##` heading), **หมวดหมู่** (เช่น คำถาม-คำตอบ / การโต้ตอบ (AR) / หมวดหมู่ใหม่), และ **รายละเอียดเกม** (อธิบายกติกา/วิธีเล่น/กลไกให้ชัดพอเอาไปทำต่อได้ — เติมทีหลังได้ถ้ายังไม่ได้คิด)
- **ถ้าไอเดียไหนมี "รายละเอียดเกม" ครบแล้ว (ไม่ใช่ "ยังไม่ได้เพิ่ม")** ให้ถามผู้ใช้ก่อนเสมอว่า "จะทำเกมนี้เลยไหม?" รอผู้ใช้ตอบ **"cf"** ก่อนจึงจะเริ่มลงมือทำ
- หลังได้ "cf" แล้ว ให้อ่านรายละเอียดเกมนั้นอีกครั้งอย่างละเอียด ถ้ามีจุดไหนไม่ชัดเจน/สงสัย/ขาดข้อมูลที่ต้องตัดสินใจ ให้ **ถามผู้ใช้ก่อนลงมือสร้างเกม** อย่าเดาเอง — ถ้าข้อมูลครบและไม่มีข้อสงสัยแล้ว ให้เริ่มทำเกมได้เลยโดยไม่ต้องถามซ้ำ

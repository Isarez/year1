# นกฮูกสนุกคิด เตรียมสอบ ป.1

เว็บแอปฝึกทำแบบทดสอบเตรียมความพร้อมสำหรับเด็กก่อนเข้าเรียน ป.1 ครอบคลุมหมวดหมู่:

- 🔢 คณิตศาสตร์
- 📖 ภาษาไทย 1
- 📝 ภาษาไทย 2 (ปลดล็อกหลังทำภาษาไทย 1 จบ)
- 🔤 English
- 🌟 พฤติกรรมดี-ไม่ดี
- 🐾 สัตว์และที่อยู่
- 📅 วันในสัปดาห์
- 🧩🔍💡🎯 เชาวน์ปัญญา 1-4 (ปลดล็อกตามลำดับ)

และหมวดเกมโต้ตอบแบบจับคู่นิ้วมือ (AR):

- 🖐️ ต่อประโยค (ไทย) — ลากคำมาเรียงในช่องให้ถูกลำดับ
- 🤟 ต่อประโยค (Eng) — ลากคำมาเรียงประโยคภาษาอังกฤษ
- 🔢 คิดเลข — โจทย์บวก/ลบเลข 1-2 หลัก ลากการ์ดคำตอบที่ถูกใส่ช่อง
- 🔗 โยงเส้น (ไทย/Eng) — จีบนิ้ว/ลากเส้นจากสัญลักษณ์ด้านซ้ายไปยังคำตอบด้านขวาให้ตรงคู่กัน มีทั้งเวอร์ชันคำตอบภาษาไทยและภาษาอังกฤษ
- 🧺 หยิบให้ครบ — อ่านโจทย์ (เช่น "หยิบแอปเปิ้ล 5 ลูก") แล้วจีบนิ้วหยิบของที่ตรงกับโจทย์จากของหลายชนิดที่ปนกันบนจอ ไปวางในตะกร้าคำตอบให้ครบจำนวน (เล่นได้เฉพาะแท็บเล็ต/คอมพิวเตอร์ เพราะต้องการพื้นที่จอกว้าง)

และหมวดเกมฝึกทักษะ (ไม่ใช้กล้อง เล่นได้ทุกอุปกรณ์):

- 🎲 จับคู่โดมิโน — เกมความจำกดพลิกการ์ด จับคู่การ์ดตัวเลขกับการ์ดโดมิโนที่มีค่าจุดเท่ากัน (1–12 จุด รูปแบบโดมิโนจริง)
- 🦜 จับคู่สัตว์ — เกมความจำเช่นเดียวกัน แต่ฝั่งซ้ายเป็น emoji สัตว์ ฝั่งขวาเป็นคำศัพท์ภาษาอังกฤษ (16 ชนิดสัตว์, 3 ด่าน)
- 🔦 ทายเงา 1 — ดูเงาสีดำแล้วเลือกภาพสีจริงที่ตรงกับเงา (สัตว์/ผลไม้/สิ่งของ 45 ชิ้น) 15 ด่าน ตัวเลือกเพิ่มจาก 3 → 5 ตามด่าน ตัวหลอกเงาคล้ายโจทย์ ตอบถูกเงาเฉลยเป็นภาพสี ตอบผิดเฉลยแล้วไปด่านต่อไป
- 👥 ทายเงา 2 — เงาดำ 2 อย่างซ้อนเหลื่อมกัน เลือกคู่ object ที่ตรงกับเงาทั้งสอง (ปลดล็อกเมื่อผ่านทายเงา 1) ตัวหลอกคละแบบ ชิ้นแรกจริง-หลังหลอก / หลังจริง-แรกหลอก / หลอกหมด โจทย์สุ่มผสมข้ามกลุ่มเล่นซ้ำไม่ซ้ำเดิม
- 🎭 ทายเงา 3 — เงาดำ 3 อย่างซ้อนกัน เลือกชุด 3 object ให้ครบถูก (ปลดล็อกเมื่อผ่านทายเงา 2) ท้าทายสุดในตระกูลทายเงา
- 🎧 ฟังคำศัพท์ 1 — ฟังคำศัพท์ภาษาอังกฤษ 3 ตัวอักษรจาก Web Speech API แล้วลากตัวอักษรมาประกอบคำในช่องคำตอบ (เฉลยบางตัวอักษรให้ก่อน, 10 ด่าน)
- 👂 ฟังคำศัพท์ 2 — เหมือนฟังคำศัพท์ 1 แต่ไม่มีเฉลยให้เลย ต้องหาตัวอักษรเองทั้งหมด (ปลดล็อกหลังทำฟังคำศัพท์ 1 จบ)
- 🗣️ ฟังคำไทย 1 — เหมือนฟังคำศัพท์ 1 แต่ฟังคำภาษาไทย 3-5 ตัวอักษรแทน (เฉลยบางตัวอักษรให้ก่อน, 10 ด่าน) ถ้า browser ไม่รองรับเสียงพูดไทยจะโชว์สัญลักษณ์คำใบ้แทนพร้อมแจ้งเตือน
- 🔊 ฟังคำไทย 2 — เหมือนฟังคำไทย 1 แต่ไม่มีเฉลยให้เลย ต้องหาตัวอักษรเองทั้งหมด (ปลดล็อกหลังทำฟังคำไทย 1 จบ)

## ฟีเจอร์

- เลือกโปรไฟล์เด็กได้หลายคน พร้อม avatar และชื่อของตัวเอง
- สุ่มสลับตัวเลือกคำตอบทุกครั้งที่ทำแบบทดสอบ ไม่ให้จำตำแหน่งคำตอบได้
- เก็บสติกเกอร์ตามหมวดที่ทำครบ พร้อมสมุดสติกเกอร์ — สติกเกอร์เป็นวงกลมทองมุมพับพร้อมเงาและแสง shine, icon เกมเอียง 30° แบบสติกเกอร์จริง
- icon ประจำหมวดเกมทั้ง 23 หมวดและหัวข้อ section เป็นภาพ SVG สีสันสดใส (แทน emoji เดิม) รวมถึง icon ปุ่มบน menu bar
- นกฮูกมาสคอต SVG ให้กำลังใจระหว่างทำแบบทดสอบ พร้อม animation เคลื่อนไหว
- เพลงพื้นหลัง 5 เพลง เล่นสลับวนกันไปเรื่อยๆ และเอฟเฟกต์เสียง เปิด/ปิดได้จากปุ่มด้านบน พร้อมเสียงแสดงความยินดีตอนจบเกม
- โหมดกลางวัน/กลางคืน สลับได้จากปุ่มด้านบน พร้อม animation ท้องฟ้าและพระอาทิตย์/พระจันทร์ไล่สีนุ่มนวล
- เกมโต้ตอบแบบจับคู่นิ้วมือผ่านกล้อง (AR) ลากการ์ดคำ/ตัวเลขหรือลากเส้นด้วยมือจริงเพื่อฝึกต่อประโยค คิดเลข และโยงเส้นจับคู่
- เกมฝึกทักษะแบบกดพลิกการ์ด (ไม่ใช้กล้อง) ฝึกความจำและการจับคู่ตัวเลข-จำนวน และจับคู่รูปสัตว์กับคำศัพท์อังกฤษ
- หน้าเลือกหมวดมี animation hover ยกการ์ด + shimmer แสง + wiggle emoji
- เกม AR ทุกโหมดสามารถหยิบการ์ดออกจากช่องคำตอบเพื่อเปลี่ยนใหม่ได้ และปล่อยการ์ดตรงไหนก็อยู่ตรงนั้น (ไม่รีเซ็ตตำแหน่ง)
- เกม AR มีปุ่มเปิด-ปิดกล้องกลางเกม (ไม่แสดงบนมือถือ) สลับไปลากด้วยนิ้ว/เมาส์ได้โดยไม่ต้องออกจากด่าน และสุ่มด่านไม่ซ้ำกันภายใน 1 รอบเล่น
- มือการ์ตูนในเกม AR วาดสดจากตำแหน่งมือจริงทุกเฟรม (ไม่ใช่ภาพนิ่ง) ขยับนุ่มลื่นไม่สั่น มีเงาใต้มือ ผิวไล่เฉดมีมิติ ปลอกแขนเสื้อฟ้าพาสเทล เล็บน่ารักๆ และจุดบอกตำแหน่งนิ้วแบบลูกแก้วเงาวาว
- เลือก/แก้ไข avatar ได้ 48 แบบ พร้อม scrollbar ในกล่องเลือก และแก้ไขชื่อ/avatar ของโปรไฟล์ที่มีอยู่แล้วได้ทั้งจาก header และหน้าเลือกโปรไฟล์
- **ย้ายข้อมูลเด็กข้ามอุปกรณ์** — Export โปรไฟล์+ความคืบหน้าเป็นไฟล์เข้ารหัส แล้ว Import บนอุปกรณ์อื่นได้ทันที (รองรับจัดการชื่อซ้ำด้วย modal Replace/Rename)
- รองรับโหมดเต็มหน้าจอ และติดตั้งเป็นแอป (Add to Home Screen) ได้บนมือถือ/แท็บเล็ต
- ข้อมูลความคืบหน้าทั้งหมดเก็บไว้บนเครื่องผู้ใช้เท่านั้น (`localStorage`) ไม่มีการส่งไปยังเซิร์ฟเวอร์ใดๆ
- ปุ่ม "รายละเอียดการอัพเดท" ในหน้าเลือกโปรไฟล์เด็ก เปิดดูสรุปสิ่งที่เปลี่ยนแปลงล่าสุดของแอปแบบอ่านง่าย

เป็น static site ล้วนๆ (ไม่มี build step) แยกไฟล์เป็น `index.html`, `css/style.css`, `js/data.js`, `js/owl-messages.js`, `js/app.js` เปิดใช้งานได้ทันทีในเบราว์เซอร์

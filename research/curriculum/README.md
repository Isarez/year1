# คลังความรู้หลักสูตร ป.1-6 (เพื่อพัฒนาเกมต่อ)

โฟลเดอร์นี้เป็นผลสรุปจากการ research หลักสูตรแกนกลางการศึกษาขั้นพื้นฐานของกระทรวงศึกษาธิการ ระดับชั้นประถมศึกษาปีที่ 1-6 (อายุประมาณ 6-12 ปี) แยกตามวิชาและระดับชั้น เพื่อใช้เป็นข้อมูลอ้างอิงตอนคิดโจทย์/เกมใหม่ให้ Owlkids ในอนาคต — **ปัจจุบันแอปยังคงเน้นระดับเตรียมสอบเข้า ป.1 (อายุ ~5-6 ปี) เป็นหลัก** เอกสารชุดนี้คือข้อมูลเผื่อขยายขอบเขตแอปในอนาคต ยังไม่ใช่แผนพัฒนาที่ตัดสินใจแล้ว

## โครงสร้างโฟลเดอร์

```
research/curriculum/
├── README.md                    ← ไฟล์นี้ (สารบัญ + วิธีอ่าน)
├── แนวคิด-mechanic-ใหม่.md        ← สรุปไอเดีย mechanic เกมใหม่ที่ควรพัฒนา เรียงตามความคุ้มค่า (ใช้ซ้ำได้หลายวิชา)
├── ป1/ ... ป6/                  ← 1 โฟลเดอร์ต่อ 1 ระดับชั้น
│   ├── คณิตศาสตร์.md
│   ├── ภาษาไทย.md
│   ├── ภาษาอังกฤษ.md
│   ├── วิทยาศาสตร์.md
│   └── สังคมศึกษา.md
```

แต่ละไฟล์วิชา/ชั้น มีโครงสร้างเดียวกัน: (1) แหล่งอ้างอิง (2) ตัวชี้วัดแยกตามสาระ พร้อมคำอธิบายภาษาง่ายๆ (3) ไอเดียโจทย์/เกมที่เป็นไปได้ต่อแต่ละหัวข้อ (4) สรุปว่าทับซ้อนกับเนื้อหาที่แอปมีอยู่แล้วแค่ไหน หรือเป็นพื้นที่ใหม่ทั้งหมด

## ความน่าเชื่อถือของแหล่งข้อมูล (สำคัญ อ่านก่อนใช้อ้างอิง)

| วิชา | แหล่งที่มา | ความมั่นใจ |
|---|---|---|
| **คณิตศาสตร์** | อ่านจากเอกสารทางการ (สสวท./สพฐ.) โดยตรงทุกตัวชี้วัด | สูงมาก — verbatim จากต้นฉบับ |
| **วิทยาศาสตร์** | อ่านจากเอกสารทางการ (สสวท./สพฐ.) โดยตรงทุกตัวชี้วัด ทุกชั้น | สูงมาก — verbatim จากต้นฉบับ |
| **ภาษาอังกฤษ** | อ่านจากเอกสารทางการ (สพฐ.) โดยตรง มีบางจุดใน ป.2-3 ที่ตัวอักษรอ่านไม่ครบ (ปัญหาไฟล์ PDF ต้นฉบับ) และเสริมจากรูปแบบที่สม่ำเสมอของชั้นข้างเคียง | สูง |
| **ภาษาไทย** | ไฟล์ PDF ต้นฉบับ (`pdftotext`) เสียหาย (font เข้ารหัสผิด อ่านไม่ออก) — แก้ด้วยการแปลงแต่ละหน้าเป็นภาพ (`pdftoppm` 120 DPI) แล้ว OCR ด้วย `tesseract` ภาษาไทย อ่านตัวชี้วัดจริงทุกข้อจากเอกสาร "ตัวชี้วัดและสาระการเรียนรู้แกนกลาง กลุ่มสาระการเรียนรู้ภาษาไทย (ฉบับปรับปรุง 2560)" ของ สพฐ. โดยตรงแล้ว (80 หน้า) | สูง — OCR จากเอกสารทางการโดยตรง มีโอกาสคลาดเคลื่อนเล็กน้อยจาก OCR โดยเฉพาะจุดตาราง/เลขไทยกำกับชั้น (ต้องยืนยันด้วยการไล่ลำดับตัวชี้วัด ไม่ใช่การอ่านมนุษย์ตรงๆ 100%) |
| **สังคมศึกษา** | ไฟล์ PDF ต้นฉบับ (173 หน้า รวม ป.1-ม.6) เสียหายแบบเดียวกับภาษาไทย — OCR ด้วยกระบวนการเดียวกัน อ่านตัวชี้วัดจริงทุกข้อจากเอกสาร "ตัวชี้วัดและสาระการเรียนรู้แกนกลาง กลุ่มสาระการเรียนรู้สังคมศึกษา ศาสนา และวัฒนธรรม (ฉบับปรับปรุง 2560)" ของ สพฐ. โดยตรงแล้ว (สาระที่ 1 ศาสนาฯ สรุปแก่นสำคัญแทนรายละเอียดเชิงศาสนา/บาลีปลีกย่อยทั้งหมด เพื่อความเหมาะสมกับการออกแบบเกม ไม่ใช่เพราะ OCR อ่านไม่ออก) | สูง — OCR จากเอกสารทางการโดยตรง มีโอกาสคลาดเคลื่อนเล็กน้อยจาก OCR โดยเฉพาะจุดตาราง/เลขไทยกำกับชั้น (ต้องยืนยันด้วยการไล่ลำดับตัวชี้วัด ไม่ใช่การอ่านมนุษย์ตรงๆ 100%) |

**ไฟล์ดิบต้นฉบับ**: PDF ต้นฉบับของคณิต/วิทยาศาสตร์/อังกฤษ (ที่เสีย, อ่านด้วย `pdftotext`) และของภาษาไทย/สังคมศึกษา (ที่เสีย, อ่านด้วย OCR ภาพหน้า 120 DPI) อยู่นอก repo ในสแครชแพดชั่วคราวของเซสชัน ไม่ได้ถูกเก็บไว้ถาวร — ถ้าต้องการอ้างอิงกลับไปที่ตัวชี้วัดต้นฉบับแบบคำต่อคำ ให้ไปดาวน์โหลด PDF จาก academic.obec.go.th ใหม่ (ใช้โดเมนเปล่าไม่มี `www.` เพราะ `www.` fail TLS cert) — ไฟล์สังคมศึกษาต้องดาวน์โหลดผ่านลิงก์ Google Drive ที่หน้านั้นชี้ไปแทน (ไม่ได้ host ตรงบน academic.obec.go.th)

## หมายเหตุอื่นๆ

- เนื้อหาทุกวิชา/ทุกชั้น เขียนโดยตั้งใจให้ "แปลง" ตัวชี้วัดทางวิชาการเป็นไอเดียโจทย์เกมที่เข้ากับสไตล์ static-page/localStorage ของแอปนี้ได้จริง (ปรนัย, drag-drop, จับคู่, เรียงลำดับ ฯลฯ) ไม่ใช่แค่ก็อปตัวชี้วัดมาแปะเฉยๆ
- ทุกไฟล์มีหมายเหตุปิดท้าย เปรียบเทียบกับหมวดหมู่ (`CATS`) ที่แอปมีอยู่แล้วตอนเขียน (กรกฎาคม 2026) — ถ้าแอปเพิ่มหมวดใหม่ไปแล้วในอนาคต หมายเหตุนี้อาจไม่ทันสมัย ให้เช็ค `js/data.js` จริงเทียบเคียงด้วยเสมอ
- ระดับความยาก/นามธรรมของเนื้อหาเพิ่มขึ้นชัดเจนตั้งแต่ ป.4 เป็นต้นไป (เศษส่วน-ทศนิยม, ไวยากรณ์อังกฤษจริงจัง, วงจรไฟฟ้า, ประวัติศาสตร์นามธรรม) — ถ้าจะทำเกมสำหรับชั้นเหล่านี้ ควรทบทวนธีม/ดีไซน์ที่เคย "เหมาะกับเด็ก 5 ขวบ" ตามหลักการออกแบบเดิมของแอป ว่ายังเหมาะกับเด็ก ป.4-6 (10-12 ปี) หรือควรปรับ
